# Lyrics Scroller

Application web simple pour faire défiler des paroles de chansons avec :

- ajout d’une chanson avec titre, artiste, paroles,
- sauvegarde locale dans le navigateur via localStorage,
- vitesse de défilement configurable,
- pause de démarrage configurable pour une introduction musicale.

## Lancer localement

Depuis le dossier du projet :

```bash
python3 -m http.server 8000
```

Puis ouvrez http://localhost:8000 dans votre navigateur.
